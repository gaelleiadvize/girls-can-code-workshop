## Setup

Before you can run or deploy the sample, you need to do the following:

1.  Install npm globaly

[See here how to install npm (Windows/Mac/Linux)][npm]


2.  Go to root app directory & run :

        npm install
3.  run 
        
        npm run start-dev
    or
        
        npm run start
